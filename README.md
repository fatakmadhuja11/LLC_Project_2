# 🧾 Invoice Generator

## 📌 Project Description
The Invoice Generator is a web-based application built using HTML, CSS, and JavaScript. It allows users to create invoices by adding client details and item information. The system automatically calculates totals, applies tax and discounts, and displays a complete invoice in a structured format.

## ❗ Problem Statement
Manual invoice creation is time-consuming and prone to errors in calculations. This project solves the problem by automating invoice generation, ensuring accuracy and saving time.

## 🚀 Features

### 🔹 Core Features
- Add client details (name, contact)  
- Add item details (name, quantity, price)  
- Automatic total calculation  
- Display invoice in structured format  

### 🔹 Advanced Features
- Tax percentage calculation  
- Discount support  
- Automatic invoice number generation  
- Dynamic invoice preview  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

## ▶️ How to Run the Project
1. Download or clone the repository  
2. Open the `index.html` file in any web browser  
3. Start generating invoices  

## 🎯 How to Use
- Enter client details  
- Add items with quantity and price  
- Enter tax and discount percentages  
- Click "Generate Invoice"  
- View final invoice with total amount
